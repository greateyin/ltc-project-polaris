# 5. 推動路徑與 KPI (Roadmap & KPIs) - Detailed

## 5.1 詳細時程表 (Gantt Chart Info)

### Phase 1: Pilot Run (試辦期)
*   **時間**: 2026/01/01 - 2026/03/31
*   **地點**: 臺北市 (聯醫系統)、新北市 (雙和醫院)
*   **里程碑**:
    *   2026/01/15: 系統規格定案 (Spec Freeze).
    *   2026/02/01: API Gateway 上線, 與 2 家醫院完成連通測試.
    *   2026/03/01: 正式上線 (Go-Live), 首筆 FHIR 轉介單成功接收.
    *   2026/03/31: 達成 100 筆轉介, 平均媒合時間 < 3 天.

### Phase 2: Smart Rental Launch (租賃新制上線)
*   **時間**: 2026/04/01 - 2026/06/30
*   **目標**: 準備迎接 7/1 新制.
*   **里程碑**:
    *   2026/04/15: 完成輔具租賃額度試算引擎 (Logic Module).
    *   2026/05/01: 與 Top 3 輔具租賃商 (e.g., 杏一, 維康) 完成 API 串接.
    *   2026/06/15: 全系統壓力測試 (Stress Test).
    *   2026/07/01: **Feature Go-Live** (配合政策同步啟動).

### Phase 3: Nationwide Scale (全國推廣)
*   **時間**: 2026/07/01 - 2026/12/31
*   **目標**: 擴展至非六都及偏鄉.
*   **里程碑**:
    *   2026/09/01: 導入 AI 需求預測模型.
    *   2026/12/31: 全國覆蓋率達 80% 醫院.

## 5.2 成功驗收指標 (KPIs & Acceptance)

| 構面 | 指標名稱 | 定義 | MVP 目標 | 年度目標 | 資料來源 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **政策** | **D2S Time** | Discharge-to-Service 平均天數 | < 5 天 | < 2 天 | SLA Agent |
| **體驗** | **Match Rate** | 首次媒合成功率 (無需人工換人) | > 70% | > 85% | Match Agent |
| **重症** | **Severe Cov** | CMS 7-8 級案件媒合成功率 | > 80% | > 95% | Match Agent |
| **租賃** | **Rental Utl** | 出院個案使用租賃輔具之比率 | N/A | > 20% | Resource Agent |
| **系統** | **Uptime** | 系統正常運作時間 | 99.5% | 99.9% | Monitoring |
